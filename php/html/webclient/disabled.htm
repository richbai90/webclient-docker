The Supportworks Webclient is currently disabled. To enable the webclient please follow the instructions outlined below against the Supportworks Server.

Open the configuration file SwServerService.xml (located in ...\Program Files\Hornbill\Supportworks Server\conf) in a text editor and look for the DataDictionaryExport section, as shown here: 
<DataDictionaryExport>
       <Enabled value="0"/>
            .
            .
</DataDictionaryExport>
Then change the Enabled value to "1". Save the configuration file and restart the Supportworks Server Service.
