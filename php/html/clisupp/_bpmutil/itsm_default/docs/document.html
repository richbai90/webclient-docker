The Business Process Import/Export Utility User Guide

Introduction

The Business Process Import/Export Utility can be used to export Business Processes from Supportworks Database into a XML file and import Business Processes from a XML file into Supportworks Data.

This utility is launched from within the File > Manage Settings > Process Settings tab of the Supportworks Client - see the Supportworks ITSM Enterprise Administration Guide for more information.

Functions available

Utility functions available include:

  • Export: Export one or more processes from Supportworks, generating an XML file
  • Manage Files: upload, download or delete generated XML files
  • Import: Import one or more generated XML files into Supportworks as processes.

Prerequisites

You will need an ODBC connection for connecting to the Supportworks Server.

Generation of primary keys

During the import new primary keys will be created for some tables (For example, pk_auto_id within the stage table will be automatically generated).

These automatically generated primary keys exist as foreign keys in related tables.

During the import these foreign keys will be updated with the values of the newly generated primary keys.
